# 認識 迴圈

迴圈是電腦科學運算領域的用語，也是一種常見的控制流程。迴圈是一段在程式中只出現一次，但可能會連續執行多次的程式碼。迴圈中的程式碼會執行特定的次數，或者是執行到特定條件成立時結束迴圈，或者是針對某一集合中的所有項目都執行一次。在一些函式程式語言中會使用遞迴或不動點組合子來達到迴圈的效果，其中尾端遞迴是一種特別的遞迴，很容易轉換為疊代。
